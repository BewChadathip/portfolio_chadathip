Project Name

Develop a model to classify sentiment from user opinions

Project Details

Develop a model for sentiment classification using the LinearSVC (Support Vector Machine) model as educational content for a course offered by the company to the general public interested in Data Science. This project involves techniques such as data cleaning, text preprocessing, features engineering and training the LinearSVC (SVM) model. Additionally, write a blog on Medium to explain the basic principles for those interested in this type of work, and create a website for users to try out the model via Streamlit.
