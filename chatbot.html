Project Name

Career Path Interest-Based Course Recommendation Chatbot

Project Details

Create a web application to display all restaurants within a university to address the issue of external visitors who come for business but are unaware of the available dining options, as well as students unfamiliar with the university's restaurants. Users can filter restaurants based on their preferences, such as à la carte restaurants, noodle shops, etc., or religious requirements like Halal food, which is scarce in the university. The system will show the restaurant name, location, menu items, allowing users to make informed decisions, along with the rating of each restaurant.
